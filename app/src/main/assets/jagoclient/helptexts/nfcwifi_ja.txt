NFCによるWi-Fiダイレクト接続

ペアリング未設定のときはメッセージ欄が黄色になっています
両方のデバイスで黄色の画面を表示した状態で
デバイスNFCタグ部を 10cm 以内に接近させて
"タップしてビーム"が表示されたら一方のデバイスの画面を
タップしてください
ペアリング設定済みのときはメッセージ欄が白色になっていて
接続、ペア解除などのボタンが表示されます
ペア設定の結果、どちらがサーバー側になるかは不確定です

設定 : 
  NFC :
    システムの"設定"の該当パネルを開きます
    必要なら、NFCを有効にする,機内モードのOn/Off(WiFiのリセット)
    などの設定が行えます
    端末によって NFC を備えていない可能性もあります
  WiFi : 
    システムの"設定"の該当パネルを開きます
    必要なら、アクセスポイントの接続、Wi-Fi Direct ピアデバイスの
    設定、デバイス名の変更などが行えます
この端末 :
  デバイス名、MACアドレス、デバイスのステータスを表示
IP Address :
  接続済みの場合表示されます
相手 :
  ペア設定済みの場合
  デバイス名、MACアドレス、デバイスのステータスを表示
IP Address :
  接続済みの場合表示されます

ポートを開く : (サーバー側)
  ペアリング済みで セッションが接続されていないとき
  クライアント側の"接続"より先に押します
接続 : (クライアント側)
  ペアリング済みで セッションが接続されていないとき
  サーバー側の"ポートを開く"の後に押します
ペア解除 :
  別のデバイスと新しい接続をする場合現在のペアリングを解除
  してください
OK :
  セッションが接続済みのとき"パートナー接続"画面に移行します
